Properties
  Name Description
chosenRoute
It's possible to inform a FlightOption from an external source. When informed, the calculation of schedules and costs will take these values into account, instead of the Rome2rio flight results.
chosenStay

It's possible to inform an hotel data in the request.

The system will try to calculate Taxi fares for the amount of days that the trip takes and sum it in the response.

Estimated amout spent in meals per day.
