Namespace Description
Door2DoorCore
Builds a complete itinerary with schedules and pricing informaition based on the request.
Door2DoorCore.Exceptions
Custom Exceptions for proper handling
Door2DoorCore.Types.Door2DoorRequest
Request data to get the door to door routes
Door2DoorCore.Types.Door2DoorRequest.OuterFlightOption

It's possible to inform a FlightOption from an external source.

When informed, the calculation of schedules and costs will take these values into account, instead of the Rome2rio flight results.
Door2DoorCore.Types.Door2DoorRequest.OuterHotelOption

It's possible to inform an hotel data in the request.

The system will try to calculate Taxi fares for the amount of days that the trip takes and sum it in the response.
